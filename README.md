# Super-awesome-hackpad
I am absolutely horrible with github, sorry if there is something wrong with this repository. But anyways<br> this this is a hackpad i've worked on for like a week?<br>

<b>FEATURES :</b>

*✩ ‎ ‎ Shortcut Modes (switch between)*<br>
*✩ ‎ ‎ 4 macros*<br>
*✩ ‎ ‎ OLED screen displaying mode and<br> ‎ ‎ ‎ ‎‎ ‎ ‎  volume adjustments*<br>
*✩ ‎ ‎ A button to change mode*<br>
*✩ ‎ ‎ star cutout for the PCB to show through*<br>
*✩ ‎ ‎ Rotary encoder (adjusts volume and mutes when pressed)*<br>
